Pesquisa para Desenvolvimento de Sistema de Gestão para Igrejas

Esta pesquisa tem como objetivo entender melhor as necessidades das igrejas em relação à organização administrativa e gestão de membros. Suas respostas ajudarão a criar uma ferramenta realmente útil para o ministério.

1. Informações da Igreja
2. Gestão Atual da Igreja
Pode marcar mais de uma.
3. Problemas Enfrentados
4. Funcionalidades Desejadas
Pode marcar várias.
5. Interesse no Sistema
6. Feedback Final

Publicação rápida no GitHub Pages: suba este arquivo como index.html em um repositório público e ative o GitHub Pages nas configurações do repositório. Para salvar as respostas de verdade, conecte este formulário a um backend como Google Apps Script, Formspree, Supabase ou Firebase.
